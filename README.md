# Dashboard-de-An-lisis-Financiero-en-Power-BI
Dashboard de BI desarrollado en Power BI para el seguimiento de KPIs financieros (Ingresos, Gastos, Margen). Permite análisis multidimensional mediante filtros interactivos. Incluye medidas DAX personalizadas y formato condicional avanzado para la visualización clara de objetivos de rentabilidad y desempeño por país, marca y segmento.
